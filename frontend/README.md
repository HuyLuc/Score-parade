# Score Parade Frontend

Frontend React + TypeScript cho hệ thống chấm điều lệnh tự động.

## Tech Stack

- React 18
- TypeScript
- React Router

## Setup

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm start

# Build production
npm run build
```

## Cấu trúc

- `src/views/` - 9 màn hình chính
- `src/components/` - Reusable components
- `src/services/` - API services

## Views

1. RegisterView - Đăng ký
2. LoginView - Đăng nhập
3. ListOfCandidatesView - Danh sách thí sinh
4. CreateCandidateView - Tạo thí sinh
5. ConfigurationView - Cấu hình
6. BaremView - Barem điểm
7. ObservationView - Chấm thí sinh
8. EndOfSectionView - Kết thúc phần
9. SummaryView - Tổng kết
